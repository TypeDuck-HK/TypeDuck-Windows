## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Backend metadata -> TSF registration | Backend `ime.json` is no longer trusted to define the required TypeDuck Windows profile. |
| Planning identity contract -> production source | Proposed GUIDs and bilingual display strings become OS-visible COM/TSF identity. |
| Build output -> installer/system directories | CMake output names must match what SetupHelper and Inno later deploy/register. |
