## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| host-to-VM | Installer and scripts are copied from the developer host into a disposable VM. |
| installer-to-OS | Elevated installer/setup helper writes system DLLs, COM/TSF registry keys, startup entries, and scheduled tasks. |
| evidence-to-repo | VM results are serialized into committed or reviewable evidence files. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-01-01 | Tampering | installer artifact evidence | mitigate | Record SHA-256, byte size, path, generated-at UTC, and command evidence for the exact `typeduck-windows-ime-setup.exe`. |
| T-07-01-02 | Spoofing | VM target selection | mitigate | Evidence records VM name, Windows version, checkpoint identity, and explicitly states the installer was not run on the host. |
| T-07-01-03 | Information Disclosure | visual/VM evidence | mitigate | D-01 through D-04 prohibit screenshot files; evidence records notes and deterministic checks only. |
| T-07-01-04 | Repudiation | install/uninstall outcome | mitigate | Preserve installer/uninstaller exit codes, log paths, registry/file/task snapshots, and timestamped command evidence. |
| T-07-SC | Tampering | package-manager installs | accept | This plan adds no npm, pip, or cargo installs. |
</threat_model>
