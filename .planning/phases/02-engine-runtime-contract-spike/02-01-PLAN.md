## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub release archive -> local runtime cache | External binary archive becomes local engine dependency. |
| Git submodule -> staged runtime | External schema repository supplies YAML, dictionaries, OpenCC assets, and plugin data. |
| Staging script -> local filesystem | Script writes runtime files and evidence under caller-provided roots. |
